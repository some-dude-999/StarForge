üîç How Duplicate Detection Works

The tool identifies duplicates in two ways:

  • Exact Size Match: Files with identical file sizes (in bytes)
  • Name Similarity: Files with very similar names (e.g., "model_v1.stl" and "model_v2.stl")

For each group of duplicates:

  • The first file is marked in GREEN (the "keeper")
  • Additional files are marked in YELLOW (candidates for deletion)
