STL Quality Checker & Duplicate Finder

What is this?

Finds and removes duplicate STL files. Saves storage space and cleanup time.

How to use

  1. Upload files - Drag & drop STL files or click to select
  2. Review auto-detected duplicates - Yellow = duplicate, Green = keep, Red = bad file
  3. Click thumbnails to view 3D models (optional but recommended)
  4. Mark files - Click "Duplicate" or "Bad File" buttons to mark for deletion
  5. Export CSV - Download list of files to delete
  6. Delete files from your computer using the CSV

Key points

  • Duplicates detected by file size and name similarity
  • First file in each duplicate group = keeper (green)
  • Always check 3D preview before deleting
  • Use arrow keys to browse models quickly

Ano ito?

Hanapin at tanggalin ang mga duplicate na STL files. Makatipid ng storage space at oras.

Paano gamitin

  1. Upload files - I-drag & drop o mag-click para pumili ng STL files
  2. Review ang duplicates - Yellow = duplicate, Green = itago, Red = sira
  3. I-click ang thumbnails para tignan ang 3D model (optional pero recommended)
  4. Markahan ang files - I-click ang "Duplicate" o "Bad File" para sa delete
  5. Export CSV - I-download ang listahan ng i-dedelete
  6. Delete ang files sa computer gamit ang CSV

Importante

  • Duplicates ay nakikilala sa file size at pangalan
  • Unang file sa bawat grupo = itago (green)
  • Laging tignan sa 3D bago mag-delete
  • Gamitin ang arrow keys para mabilis na mag-browse
